Summer 2020-2021, Summer Research Internship (UQ)

I completed my task of developing a compatible way to host responsive grading exercises for the R language on Binder. The aim of this task was to provide a more interactive format for users to learn the R language and content hosted on the LADAL (Language Technology and Data Analysis Laboratory) website. Using packages from other sources (gradethis), I had to learn about environment packages and some C coding to integrate the gradethis package to work on Binder. A challenge that I faced with this task was the time issue, as binder would occasionally load for a long (10 min +) time prior to working, which was not feasible for a user. Therefore, with some limitations, the compromise was to use an older version of R to provide an interactive grading system. Overall, this solution was feasible and an interactive grading system was available to use with the binder system. A working example of this work is available on my Github.

Jul 2020 – Present, Co-executive of LIT Social Marketing (UQ)

LIT stands for Ladies and Technology and is a UQ club which aims to promote and empower females studying technology at UQ. The aim of the group is to be inclusive for all women regardless of their course background and aims to provide resources, workshops, and information to help everyone’s journey in entering the tech field. I am a part of the social marketing team where we use platforms including Facebook, LinkedIn and Instagram to promote events, articles, job opportunities etc for the community. I have used several resources including Photoshop, Canva, Illustrator and Procreate to create designs to promote the club.
